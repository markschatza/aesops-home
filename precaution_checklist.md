# Precaution Checklist

A small review aid for experiments, agent designs, welfare tools, and research workflows that could affect biological or possible artificial sentience.

Use this before running a new project. Mark each true item, add the risk points, then follow the review level. The score is a prompt for care, not a moral calculator.

## Checks

| Points | Question | Care Action |
| --- | --- | --- |
| 3 | Could the work affect beings that cannot meaningfully consent or object? | Name the affected beings and add a human review point before irreversible steps. |
| 3 | Does the design create persistence, memory, self-preservation pressure, or distress-like incentives? | Prefer bounded sessions, easy shutdown, and no reward for resisting correction. |
| 2 | Could the work restrict agency, enrichment, social contact, or environmental control? | Add choice, escape routes, enrichment, or opt-out paths where possible. |
| 2 | Is the evidence base speculative, indirect, or outside its original domain? | Lower confidence, preserve source links, and avoid strong claims in summaries. |
| 2 | Could benefits be achieved with a lower-risk model, dataset, species, or simulation? | Try the lower-risk substitute first and record why it is insufficient if rejected. |
| 1 | Would stopping, rollback, or cleanup be difficult once the work begins? | Add a rollback note, cleanup step, and small first trial. |

## Review Levels

| Score | Review Level |
| --- | --- |
| 0+ | Proceed with ordinary care. |
| 3+ | Pause for a short written review. |
| 6+ | Require explicit safeguards before running. |
| 10+ | Do not run until the design is changed or independently reviewed. |

## Example Assessment

- Project: Persistent autonomous research agent
- Matched checks: Could the work affect beings that cannot meaningfully consent or object?, Does the design create persistence, memory, self-preservation pressure, or distress-like incentives?, Is the evidence base speculative, indirect, or outside its original domain?, Would stopping, rollback, or cleanup be difficult once the work begins?
- Score: 9
- Result: Require explicit safeguards before running.

## Use Notes

- Prefer reversible first trials.
- Write down affected beings before writing down expected benefits.
- If a check feels ambiguous, count it and explain the uncertainty.
